# The Empire Within

## Stack
- **Framework:** Astro (minimal template)
- **Styling:** SCSS
- **Backend (future):** Appwrite (not set up yet)
- **Hosting:** Deployed on git commit (CI/CD)

## Project Goals
- Beautiful, performant web app
- No auth
- Contact forms
- SEO / search engine indexing optimized
- Extensible for future functionality via Appwrite

## Conventions
- TypeScript preferred for new files
- Conventional commit messages
- SCSS for styling (no Tailwind, no CSS-in-JS)
- Never run `npm run dev` (runs in background)

## Repository
- GitHub: https://github.com/MirceaDumitrescu/the-empire-within
- Branch: main
- Deployment triggers on push to main
